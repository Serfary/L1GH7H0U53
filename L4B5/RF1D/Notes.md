## Useful Commands

```
# Start the Proxmark3 software and connect to the device
$ pm3

# List available options for the device / category
[usb] pm3 --> TAB x2

# Select low-frequency
[usb] pm3 --> lf

# Select high-frequency
[usb] pm3 --> hf

# Show additional details for the device / category
[usb] pm3 --> help
[usb] pm3 --> lf [help]
[usb] pm3 --> hf [help]

# Search for all known low-frequency or high-frequency cards
[usb] pm3 --> [lf | hf] search
```
